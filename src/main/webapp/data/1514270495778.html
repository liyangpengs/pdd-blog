      由于工作需要,一直用到的数据库是Hbase，所以我就在这里记录下笔记,以后备用。。

      hbase众所周知,他是一个大数据专用的数据库,在拥有集群,并且硬盘支持的情况下 能容纳非常大的数据。

      好吧,不多闲聊!笔记楼主对大数据方面懂的也不是很多！对于Hbase安装,就不多做讲解,阁下请自行百度吧,

     下面我就直接介绍怎么用了。

    1.Hbase基本命令

    

   

     使用Java操作Hbase太简单了!就不记录了。    

    以上便是Hbase部分命令。需要的可以稍微记下功能,只在这里记录Hbase shell的使用,

    创建一张表: create 'xxxx','d'   说明:创建一张名为xxxx的表,并指明改表包含一个名字为d的列

    禁用一张表:disable 'xxxx'

    启用一张表:enable 'xxxx'

    增加一条数据:put 'xxxx','1001','d:a','字段a'  说明:在 'xxxx' 表新增一条数据,rowkey为:1001 

    为d列新增一个字段名为a,设置他的值为 '字段a'。

    查询一条数据:get 'xxxx','1001'  说明:返回结果为 '字段a'  说明:在'xxxx'表查询 Rowkey为 '1001'的数据,

    返回RowKey为1001的全部字段值。

    删除一条数据:deleteall 'xxxx','1001' 说明:删除RowKey为 1001 的这一条数据

    删除某条数据的某个字段:delete 'xxxx','1001','d:a' 说明:删除 'xxxx' 表中 RowKey为1001的'd:a'字段

    (并不删除这个RowKey的其他字段值)。

    得到表中的第一条数据: scan 'xxxx',{LIMIT=>1} 返回此表中的第一条数据

    扫描表中的单个字段: scan 'xxxx',{COLUMN=>'d:a'} 返回整个表中 d:a 列的值 

    获取单条数据修改Scan为get 正常带上Rowkey皆可。

    version 可以返回Hbase版本信息

    删除一张表:首先要先禁用此表,然后 drop 'xxxx' 则可以删除 'xxxx'表

    list 返回整个Hbase中的所有表

    count 'xxxx' 返回'xxxx'表的数据数据 一个Rowkey表示一条数据

    以上就是Hbase的基本操作,需要更加复杂的操作可以自己查找Hbase操作文档的哦！
